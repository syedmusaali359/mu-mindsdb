## Testing: OpenAI - Answering Questions without Context Tutorial

1.Testing CREATE MODEL
```sql
CREATE MODEL mindsdb.openai_test_a
PREDICT answer
USING
    engine = 'openai',
    question_column = 'question';
```

FROM mindsdb.openai_test_a
WHERE question = 'Where is Stockholm located?';
