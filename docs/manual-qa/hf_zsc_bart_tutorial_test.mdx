## Testing: Hugging Face - Zero-Shot Classification Bart Tutorial

Testing CREATE MODEL

```sql
CREATE MODEL mindsdb.hf_zs_bart
PREDICT PRED
USING
engine = 'huggingface',
task = 'zero-shot-classification',
model_name = 'facebook/bart-large-mnli',
input_column = 'text',
candidate_labels = ['Books', 'Household', 'Clothing & Accessories', 'Electronics'];
```
Status of the model

```sql
SELECT *
FROM mindsdb.models
WHERE name = 'hf_zs_bart';

```
