## Testing: Hugging Face - Industry Tutorial

1. Testing Create Model

```sql
CREATE MODEL mindsdb.hf_industry
PREDICT PRED
USING
engine = 'huggingface',
task = 'text-classification',
model_name = 'sampathkethineedi/industry-classification',
input_column = 'text';
```
