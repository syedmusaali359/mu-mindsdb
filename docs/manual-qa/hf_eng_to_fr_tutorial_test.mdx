## Testing: Hugging Face - Translation English to French Tutorial

1. Testing CREATE MODEL
```sql

CREATE MODEL mindsdb.hf_t5_en_fr
PREDICT PRED
USING
engine = 'huggingface',
task = 'translation',
model_name = 't5-base',
input_column = 'text',
lang_input = 'en',
lang_output = 'fr';

```
